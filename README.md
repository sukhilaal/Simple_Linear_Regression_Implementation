# 📈 Simple Linear Regression Implementation

## Overview

This project demonstrates the implementation of Simple Linear Regression using Python and Scikit-Learn. The notebook covers data preprocessing, model training, prediction, and visualization of results to understand the relationship between independent and dependent variables.

## Objectives

* Understand the concept of Simple Linear Regression.
* Train a regression model using Scikit-Learn.
* Visualize the regression line and predictions.
* Evaluate model performance.

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn

## Project Workflow

1. Import required libraries.
2. Load and preprocess the dataset.
3. Perform exploratory data analysis.
4. Split data into training and testing sets.
5. Train the Linear Regression model.
6. Make predictions on test data.
7. Visualize the regression line and results.
8. Evaluate model performance.

## Results

The model successfully learns the relationship between the input and output variables and provides predictions using the fitted regression line.

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/sukhlilaal/Simple_Linear_Regression_Implementation.git
```

2. Install dependencies:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

3. Open the Jupyter Notebook and run all cells.
